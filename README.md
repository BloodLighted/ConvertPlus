# Bloodify
## A overly-optimized HDify Fork
Bloodify is a fork of the HDify Roblox plugin that converts Accessories and Faces to HD, and removing bloat from said accessories.
**If you are _not_ interested in instance count optimization of Accessories and or making the Face HD, then you are free to click off.**

# Installation
## Manual
Open up any Roblox Studio project (Be it public or private) and then click on the `Plugins` tab, then click `Plugins Folder`. This brings you to the folder where local plugins are stored. Then drag the `.rbxm` into the folder and restart Studio.
## Creator Store
Either go to `https://create.roblox.com/store/plugins`, the toolbox >> plugins, or go to `create.roblox.com/store/asset/110962385067444/Bloodify-BETA`. If you use either of the first two methods, search for Bloodify, and filter for the creator `Heavenly_Strings`. Then, you would want to click Get Plugin/Install on either of the three options.
